# 邮件订阅 - 快速开始卡片

## ⚡ 30秒快速配置 (Mailchimp)

### 1️⃣ 获取凭证

```
Mailchimp 登录后:
- API Key: Account → API keys → Create Key
- Server: API Key 最后的 -xx 部分 (如: us1)
- Audience ID: Audience → Settings → Audience ID
```

### 2️⃣ 配置环境变量

编辑 `.env` (或 `.env.local`):

```env
MAILCHIMP_API_KEY=你的API密钥
MAILCHIMP_API_SERVER=us1
MAILCHIMP_AUDIENCE_ID=你的Audience ID
```

### 3️⃣ 启用功能

编辑 `data/siteMetadata.js`:

```javascript
newsletter: {
  provider: 'mailchimp',
},
```

### 4️⃣ 重启服务器

```bash
npm run dev
```

### 5️⃣ 测试

在网页底部找到订阅表单，输入邮箱测试 ✅

## 🔧 常见问题速查

### 问: 环境变量在哪设置？

```
开发环境: 编辑 .env 或创建 .env.local
生产环境 (Vercel): 项目设置 → Environment Variables
```

### 问: 如何查看订阅者？

```
- Mailchimp: Audience → 查看订阅者列表
- 其他服务: 相应服务的 Dashboard
```

### 问: 测试后没收到邮件？

```
1. 检查 .env 凭证是否正确
2. 查看浏览器控制台 F12 → Console 是否有错误
3. 检查邮箱垃圾箱
4. 检查 Mailchimp 中是否启用自动欢迎邮件
```

### 问: 如何禁用邮件订阅？

```javascript
// siteMetadata.js
newsletter: {
  provider: '',  // 留空禁用
},
```

### 问: 支持哪些邮件服务？

```
- mailchimp (推荐)
- buttondown
- convertkit
- emailoctopus
- klaviyo
- revue
```
