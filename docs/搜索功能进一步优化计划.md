# 搜索功能进一步优化计划（可选）

## 现状

- Flexsearch 已上线，支持中文分词与实时搜索。
- 键盘导航与无障碍提示已集成，用户体验良好。

## 可选优化

- 引入 Web Worker 进行索引加载与查询，降低主线程阻塞。
- 查询防抖与结果缓存（LRU）以提升连续输入性能。
- 索引分片：按分类/标签拆分索引文件，按需加载。
- 结果排名改进：结合 `title` 命中权重与最近更新时间做加权。

## 渐进实施

1. 在 `public/search.json` 保持现有格式；为 Worker 增加 `scripts/search-worker.js`。
2. 在 `components/Search.js` 引入 Worker 通道与防抖。
3. 增加简单缓存层，命中相同 query 时直接返回。

## 验证

- 37 篇文章场景下输入 3-10 字符时界面流畅无卡顿。
- Lighthouse 与 Web Vitals 维持稳定。
